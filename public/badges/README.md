# App Store Badges

## Getting Official Badges

### App Store Badge
- Download from: https://developer.apple.com/app-store/marketing/guidelines/
- Recommended size: 180x60px or higher resolution
- Save as: `appstore.png`

### Google Play Badge
- Download from: https://play.google.com/intl/en_us/badges/
- Recommended size: 180x60px or higher resolution  
- Save as: `googleplay.png`
