## Product Placeholders

The website has 3 product cards ready for your photos. Each has:
- Image placeholder (you'll replace with real photos)
- Product title
- Description
- Price
- Category badge

Just replace the placeholder `<div class="product-image"></div>` with:
```html
<img src="your-photo.jpg" alt="Vintage Lure Keychain" class="product-image" />
```
